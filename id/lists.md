# Membuat list Markdown

## Angka

```markdown
1. Padi
2. Kapas
3. Roti
```

1. Padi
2. Kapas
3. Roti

Bahkan, saat Anda menambahkan angka acak, tetaplah 1,2 dan 3 yang terlihat.

```markdown
1. Padi
8. Kapas
12. Roti
```

1. Padi
8. Kapas
12. Roti

## Karakter list

Jika anda tidak tertarik dengan angka, gunakan:

```markdown
- Padi
- Kapas
- Roti

* Padi
* Kapas
* Roti
```

- Padi
- Kapas
- Roti

* Padi
* Kapas
* Roti
